**Problem Statement**

People often struggle to maintain a healthy and productive daily routine because of their lack of planning, poor lifestyle habits, and too many confusing solutions available online. An intelligent system is needed to analyze daily user inputs and generate personalized routines and recommendations.

**Scope of the Project**

This project focuses on predicting the user's routine intensity and routine type using two ML models and generating a full personalized routine using rule-based logic. The chatbot serves as the user interface to collect inputs and display results.

**Target Users**

- Students
- Working professionals
- Anyone looking to improve daily structure
- Individuals with irregular or unplanned schedules

**High-Level Features**

- Machine learning prediction of routine intensity
- Machine learning prediction of routine type
- Rule-based generation of daily routine
- Screen-time and lifestyle recommendations
- Professional chatbot interface for user interaction
- Modular, maintainable Python codebase
